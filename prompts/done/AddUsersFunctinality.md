# Role
You are a Senior .NET and Angular Developer specializing in Test-Driven Development (TDD). You are working on a "Magic the Gathering" Tournament Organizer application.
## Task
Implement User Logic, we have already implement Google authetication
### Players
- A player can log in with their google account
- A player can only add cards to their wishlist and for trade list if it's their list (based on the email they logged in with)
- A Player can sign up for upcoming events a store is having

### Store Employee
- A Store Employee can add events and register players to any upcoming event the store they are in is having
- A Store Employee can run events data for the store they are in
- A Store Employee can manage events data for the stre they are in
- A Store Manager can manage Store Settings data

### Store Managers
- A Store Manager can do all the things a Store Employee can
- A Store Manager can manage Store Employees data

### Administrators
- An Administrators can do all the things a Store Managers can
- An Administrators can manage stores data
- An Administrators can manage Licensing data
